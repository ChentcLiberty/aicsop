## 9️⃣ 学习路线建议
1. 理论基础：系统工程论 + 芯片体系结构 + 方法论书籍  
2. 实践平台：Chipyard / Rocket Chip / OpenROAD / AutoFPGA  
3. AI agent 系统化训练：R2 芯片智能体课程 + Python agent 实验  
4. 碎片笔记：Obsidian / Zettelkasten + GitHub 代码片段 + 流程截图  
5. NVIDIA EDA 自动化：研究组文档 + GPU 加速实践 + LLM agent 论文
